Acciones de mantenimiento

Herramientas administrativas para limpiar o restablecer datos persistidos.

Limpiar operaciones Elimina ventas, compras, caja e inventario persistidos.
Restablecer catálogos base Reinicia productos, usuarios y proveedores a sus datos por defecto.
Limpiar trazabilidad Elimina auditoría, alertas descartadas y el último respaldo cacheado.
Reinicio controlado Vacía el sistema pero conserva configuración persistida.
Reinicio total Elimina toda persistencia local, incluida la configuración.
Exportar diagnóstico Descarga un JSON con resumen, hallazgos y configuración actual.
